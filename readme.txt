== Description ==
The pluging for embed youtube video using youtube video id.

= Features =
* Add a video to your wordpress site widget area.

== Installation ==
1. Upload the entire `sm-youtube-video-iframe` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now YouTube Video iFrame widget abailable in your widget section.
4. Also you have SM YouTube Video iFrame in settings section for YouTube Subscribe Button.

= Usage =

1. Go to Appearance > Widgets.
2. Enable SM YouTube Video iFrame widget.
3. Enter widget title
4. Enter video id;
5. Enter other field value if you want.
8. Hit the save button.
